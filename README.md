# Nvidia Jetson Orin Nano Super
NVIDIA Jetson Orin Nano Super Developer Kit is an embedded AI computer designed to run real neural networks locally on the device without offloading data to the cloud.<br>

AI Performance 	      -  67 INT8 TOPS<br>
GPU 	                -  NVIDIA Ampere architecture with 1024 CUDA cores and 32 tensor cores<br>
CPU 	                -  6-core Arm® Cortex®-A78AE v8.2 64-bit CPU 1.5MB L2 + 4MB L3<br>
Memory 	              -  8GB 128-bit LPDDR5<br>
Storage 	            -  Supports SD card slot and external NVMe<br>
Power 	              -  7W–25W<be>
